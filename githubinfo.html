All About Github

  • Q. What is a version control system?
  • A. Version control systems intelligently track changes in files. It is most helpful when a group of people are all making changes to the same files.

  • Q. What is GitHub?
  • A. GitHub is a cloud-based platform where you can store, share, and work together with others to write code.

  • Q. What is Git?
  • A. Git is a version control system for tracking changes in code.

  • Q. How is GitHub different from Git?
  • A. GitHub is the hosting service to store your code but Git is the tool used for managing and tracking changes in the code AKA version control.

  • Q. What is a repository?
  • A. A repository is similiar to a standard computer folder. It stores your files, code, and documents.

  • Q. What is a commit?
  • A. A commit is a saved change in a repository with a message describing the change made.

  • Q. What is a branch?
  • A. A branch is esentially a clone of the default branch or any other existing branch from the repository. It is used for testing certain ideas or features without the concern of others making any changes to it while you work on it.
