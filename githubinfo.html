All About Github

  • Q. What is a version control system?
  • A. A version control system helps track changes in code and manage different versions.

  • Q. What is GitHub?
  • A. GitHub is a cloud platform that hosts Git repositories and adds collaboration tools.

  • Q. What is Git?
  • A. Git is a distributed version control system for tracking changes in code.

  • Q. How is GitHub different from Git?
  • A. Git is the tool; GitHub is the service built around Git for collaboration.

  • Q. What is a repository?
  • A. A repository (repo) is a project folder that stores code, docs, and files.

  • Q. What is a commit?
  • A. A commit is a saved change in a repository with a message describing it.

  • Q. What is a branch?
  • A. A branch is a separate line of development in a repository.
