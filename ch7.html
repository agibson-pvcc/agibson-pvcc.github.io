Briefly describe or define:
Computer network - A computer network is a set of independent computer systems interconnected by telecommunication links for the purpose of sharing information and resources.
Analog device (Define and give an example) - A physical quantity used to represent information, usually voltage level, is continuous and can take on any value. The voice-oriented dial-up telephone network was originally an analog medium.
Digital device (Define and give an example) - A computer produces digital information - Specifically, a sequence of 0s and 1s. Digital devices include mobile phones, cameras, speakers, video game consoles, and your automobile's sound system.
Modem - A modem modulates, or alters, a standard analog signal called a carrier wave so that it encodes binary information. The modem modifies the physical characteristics of the carrier wave, such as amplitude or frequency, so that it is in one of two distinct states, one state representing 0 and the other state representing 1.
Bandwidth - A measure of the width of a frequency range.
Ethernet - In the commercial and office environment, the most widely used broadband technology is Ethernet. It was developed in the mid-1970s at Xerox PARC in Palo Alto, California.
WLAN - In a wireless local area network (WLAN), a short-distance form of wireless networking, a user transmits from their device to a local wireless base station (router or hotspot) that connects to a wired network.
IEEE 802.11 wireless network standard - One of the most widely used standards for wireless local access is Wi-Fi, officially known as the IEEE 802.11 wireless network standard.
MAN - A metropolitan area network (MAN) is a wireless network whose scope is larger than a WLAN, typically spanning a few blocks to an entire city.
Internet of Things - The Internet of Things (IoT) involves tagging and connecting objects such as appliances, vehicles, and even pets to the Internet for communication or control.
LAN - A local area network connects hardware devices such as computers, printers, and storage devices within a small geographic area.
Repeater - A repeater is a device that amplifies and forwards a signal, allowing two LANs to function as if they were a single network.
Bridge - A bridge examines every message to determine whether it should be forwarded between networks, intelligently connecting LAN segments.
WAN - A wide area network connects devices that are separated by large geographic distances, such as cities or countries.
Packet and packet switching - A packet is a unit of data transmitted over a network, and packet switching is the method of sending these packets independently to improve efficiency.
ISP - An Internet service provider is a company that provides access from a private or home network to the Internet.
Firewall - Firewalls are hardware or software systems that protect private networks from unauthorized access using filtering rules and policies.
Protocol - In networking, a protocol is a set of agreed-upon rules and conventions that govern the exchange of information between systems.
Cloud computing - Cloud computing is the concept of storing and accessing resources and data on remote servers instead of local machines.
Net neutrality - Net neutrality means Internet service providers should treat all users, content, and platforms equally, without discrimination.
Briefly describe the five layers of TCP/IP (The Internet protocol hierarchy):
Layer 1: Physical layer - Governs the exchange of binary digits across a physical channel such as fiber-optic cable, copper wire, or wireless radio.
Layer 2: Data link layer - Responsible for creating an error-free message pipe where messages are transmitted correctly and in proper sequence.
Layer 3: Network layer - Handles the addressing and delivery of messages between nodes across the network.
Layer 4: Transport layer - Assigns port numbers to programs and ensures messages are delivered to the correct application.
Layer 5: Application layer - Defines rules for end-user services such as email, web access, and file transfer built upon lower layers.
List and briefly describe at least 3 network services and benefits:
Email (electronic mail) - A fast and convenient way to send messages that can be read anytime, anywhere.
Social Networking - Online platforms that allow users to connect, share, and communicate globally.
Resource Sharing - Enables multiple users to share hardware, software, and data resources efficiently.
